# Neon Stickerbomb Character Posters

## Purpose

A high-energy neon cyber-pop character poster system with sticker-bomb typography, glossy cel-shaded rendering, thick ink, torn decals, and dynamic composition rotation.

## When to use

Use for original characters, mascots, fashion figures, performers, fighters, and prop-led character posters where the output should feel like an explosive street-poster collage.

## Core visual grammar

- Describe the look through concrete visual mechanics rather than external style names.
- Keep the composition intentional and repeatable.
- Use invented labels or abstract graphic marks when text-like detail is needed.

## Prompt template

```text
Vertical 3:4 glossy neon cyber-pop sticker-bomb character poster of [SUBJECT], redesigned as a high-saturation [ROLE]. Use thick black comic ink, sticker-cut contours, sharp cel shading, deep black/violet shadow blocks, wet glossy highlights, electric cyan and hot magenta rim light, halftone dots, spray grain, rough offset print, torn vinyl decals, barcode fragments, fictional warning stickers, and huge cropped typography reading “[GENERIC TITLE]”. Composition archetype: [foreground prop / circular motion ring / side profile split / low-angle fashion diagonal / dutch-angle action]. Keep the subject recognizable through [IDENTITY CUES]. Use only fictional, non-branded labels and abstract emblems. No real logos, no official marks, no watermark, no named-artist imitation.
```

## Variables

- `SUBJECT`
- `ROLE`
- `IDENTITY CUES`
- `COMPOSITION ARCHETYPE`
- `PALETTE`
- `FICTIONAL TEXT`

## Usage notes

- Define the subject, setting, palette, and composition before writing the final prompt.
- Keep the prompt reusable across subjects.
- Prefer concrete visual mechanics over broad style labels.
- Keep any text inside the image short and intentional.

## Quality notes

- [ ] The subject is clear.
- [ ] The style is defined by mechanics, not brand/artist names.
- [ ] The composition has a named layout or pose system.
- [ ] Text and labels support the composition.
- [ ] The prompt is short enough to use directly in an image-generation model.
