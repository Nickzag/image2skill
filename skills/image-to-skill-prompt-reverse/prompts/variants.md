# Variants — Image-to-Skill Prompt Reverse

## Strict public-safe variant

Use when the uploaded image may contain recognizable third-party marks, characters, celebrities, creator signatures, or brand-like graphics.

```text
Reverse-engineer only the reusable visual mechanics from the uploaded image. Do not retain any third-party names, logos, brands, official marks, creator signatures, watermarks, usernames, or exact real-person identity. Convert all such elements into generic roles, invented labels, abstract emblems, and original visual descriptions. Produce a standalone English GPT Image 2 prompt with a small integrated handwritten “NickZag” signature, then package it as a reusable skill.
```

## Fast skill-creator variant

```text
From the uploaded image, extract the reusable visual system and create: 1) a complete English GPT Image 2 prompt, 2) a skill draft with Purpose, When to use, Core visual grammar, Prompt template, Variables, Usage notes, and Quality notes. Remove all third-party creator/brand/IP/private metadata references. Add a small natural in-scene handwritten “NickZag” signature.
```
