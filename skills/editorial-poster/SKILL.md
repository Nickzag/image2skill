# Editorial Poster

## Purpose

A loose white-paper editorial illustration poster system with gouache/marker brushwork, rough typography, meaningful theme-only objects, and zine-like layout energy.

## When to use

Use for editorial illustration prompts about places, events, craft, food systems, recycling, community scenes, travel observations, and poetic everyday subjects.

## Core visual grammar

- Describe the look through concrete visual mechanics rather than external style names.
- Keep the composition intentional and repeatable.
- Use invented labels or abstract graphic marks when text-like detail is needed.

## Prompt template

```text
Loose digital gouache-and-marker editorial poster on clean white/off-white paper, translucent layered strokes, dry-brush streaks, visible sketch construction lines, broken colored contours, imperfect ink edges, scanned personal-zine texture. Theme: [THEME]. Every visible element must support the theme. Primary layout mode: [blue route text / distressed black serif / soft object collage / street handwritten]. Show [PEOPLE/ACTION] with theme-specific objects only: [OBJECT INVENTORY]. Typography: rough hand-rendered text “[MAIN TEXT]” plus small handwritten annotations. No unrelated motifs, no polished vector art, no photorealism, no corporate ad.
```

## Variables

- `THEME`
- `LAYOUT MODE`
- `PEOPLE/ACTION`
- `OBJECT INVENTORY`
- `MAIN TEXT`
- `PALETTE`

## Usage notes

- Define the subject, setting, palette, and composition before writing the final prompt.
- Keep the prompt reusable across subjects.
- Prefer concrete visual mechanics over broad style labels.
- Keep any text inside the image short and intentional.

## Quality notes

- [ ] The subject is clear.
- [ ] The style is defined by mechanics, not brand/artist names.
- [ ] The composition has a named layout or pose system.
- [ ] Text and labels support the composition.
- [ ] The prompt is short enough to use directly in an image-generation model.
