# Glossy Character Fashion Poster

## Purpose

A polished semi-realistic anime glamour/fashion poster system with porcelain-white skin, glossy material hierarchy, saturated simplified backgrounds, and rotating pose families.

## When to use

Use for original adult character fashion posters, modernized archetypes, performer/fighter/fantasy portraits, and consistent poster sets that need strong pose and palette variation.

## Core visual grammar

- Describe the look through concrete visual mechanics rather than external style names.
- Keep the composition intentional and repeatable.
- Use invented labels or abstract graphic marks when text-like detail is needed.

## Prompt template

```text
[SUBJECT] translated into a glossy semi-realistic anime fashion-poster style. Single adult character, vertical tight-cropped fashion illustration, modern glossy styling, simplified high-saturation character-specific backdrop. Character anchors: [HAIR], [EXPRESSION], [OUTFIT COLORS], [PROP]. Pose family: [POSE FAMILY], clearly different from recent outputs. Smooth doll-like semi-realistic anime glamour face, cold-white porcelain skin, controlled glossy highlights, satin/vinyl/metal/glass material hierarchy, one dominant clean saturated background color plus black/white neutrals. Use fictional non-branded details only. No real brand logos, no official marks, no watermark, no random text.
```

## Variables

- `SUBJECT`
- `HAIR`
- `EXPRESSION`
- `OUTFIT COLORS`
- `PROP`
- `POSE FAMILY`
- `BACKGROUND COLOR`

## Usage notes

- Define the subject, setting, palette, and composition before writing the final prompt.
- Keep the prompt reusable across subjects.
- Prefer concrete visual mechanics over broad style labels.
- Keep any text inside the image short and intentional.

## Quality notes

- [ ] The subject is clear.
- [ ] The style is defined by mechanics, not brand/artist names.
- [ ] The composition has a named layout or pose system.
- [ ] Text and labels support the composition.
- [ ] The prompt is short enough to use directly in an image-generation model.
