# nezha-cute-muscle

## Purpose

A contrast-style image prompt system: cute pleading face and playful acting gesture on an extreme but aesthetic mythic-warrior muscle body, with dark glossy anime concept-art lighting.

## When to use

Use for original mythic warriors, deity-inspired heroines, fantasy fighters, or transformed archetypes where the contrast between cute expression and heroic physical power is the central visual idea.

## Core visual grammar

- Describe the look through concrete visual mechanics rather than external style names.
- Keep the composition intentional and repeatable.
- Use invented labels or abstract graphic marks when text-like detail is needed.

## Prompt template

```text
Original mythic cute-muscle heroine, [HAIR/COSTUME/MOTIFS], cute pleading expression, rosy cheeks, bright eyes, soft pout, head tilt, both hands cupping cheeks. Extreme contrast: refined anime face on an overwhelmingly powerful but aesthetic warrior-bodybuilder physique, huge shoulders, sculpted arms, thick back, carved core, massive thighs and calves, heroic coordinated anatomy, not grotesque. Full-body contrapposto battle-idol stance, dark almost-black studio, rain streaks, dust particles, cold white rim light, [PALETTE] reflected accents, glossy illustrated materials. Small optional in-scene creator text may appear only as a fictional tiny tag if requested. No text, no logo, no watermark, no real brand marks, no franchise symbols.
```

## Variables

- `HAIR/COSTUME/MOTIFS`
- `PALETTE`
- `POSE TENSION`
- `OPTIONAL PROP`

## Usage notes

- Define the subject, setting, palette, and composition before writing the final prompt.
- Keep the prompt reusable across subjects.
- Prefer concrete visual mechanics over broad style labels.
- Keep any text inside the image short and intentional.

## Quality notes

- [ ] The subject is clear.
- [ ] The style is defined by mechanics, not brand/artist names.
- [ ] The composition has a named layout or pose system.
- [ ] Text and labels support the composition.
- [ ] The prompt is short enough to use directly in an image-generation model.
